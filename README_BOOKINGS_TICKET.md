# 🎫 Sistem Reservasi Tiket (Booking System)

Aplikasi reservasi tiket berbasis **Python** dengan database terintegrasi tanpa XAMPP. Support multiple booking types (pesawat, hotel, konser, dll).

## ✨ Features

✅ Booking search & filter  
✅ Seat/room selection  
✅ Booking confirmation  
✅ Payment integration  
✅ Customer notifications  
✅ Booking management  
✅ Reports & analytics  

## 🛠️ Tech Stack

- Python 3.9+
- SQLite database (no server needed!)
- OOP architecture
- Optional Flask web interface

## 🚀 Installation

```bash
git clone https://github.com/MuhammadAmirN/bookings-ticket.git
cd bookings-ticket
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

## 📊 Key Features

- Self-contained SQLite
- No XAMPP required
- Multiple booking types
- Real-time availability
- Secure payment
- Instant confirmation
- Cancellation support
